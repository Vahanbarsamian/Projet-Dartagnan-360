### 🚀 ÉTAPE 1 : LA PRÉPARATION (Une seule fois)
1.  **Ollama** : [ollama.com](https://ollama.com).
2.  **Android Studio** : [developer.android.com/studio](https://developer.android.com/studio).
3.  **L'Armée** : Copiez-collez les commandes `ollama pull` du tableau ci-dessus.

### 🚀 ÉTAPE 2 : LE RITUEL DE LANCEMENT
1.  **Windows** : Double-cliquez sur `Dartagnan360.bat`. 
2.  **Apple (macOS)** : Faites un clic droit sur `Dartagnan360.command` > Ouvrir.
3.  **Interface** : Cliquez sur **🚀 LANCER LE SYSTÈME**.

⚠️ **NOTE SUR LES ERREURS** : Si un expert affiche "Erreur", c'est presque toujours dû à un manque de mémoire (VRAM) de votre carte graphique. Pour les calculs lourds, ne cochez qu'**un seul expert à la fois**.

*   **JOB SEEKERS**: Analyze offers and optimize technical code.

### 🚀 SETUP & LAUNCH
1.  **Install** Ollama and Android Studio.
2.  **Recruit** your army using `ollama pull [model_name]` from the table above.
